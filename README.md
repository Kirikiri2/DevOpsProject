## Проект по DevOps

> Проект представляет собой сайт, на котором представлены портфолио участников команды. Цель нашего проекта - креативно осветить сильные стороны участников.

В данной работе мы практикуем навыки командной разработки, работу с фрейморками Next.js, Tailwind CSS, а также с компонентными библиотеками и библиотеками для анимаций. Вся работа руководится и координируется посредством системы котроля версий Git с помощью сервиса GitHub.

#### Git

Для нашего проекта мы выбрали систему разделения прав, где есть тимлид, отвечающий за архитектуру репозитория, документацию и подтверждение Pull Request разработчиков, работает в ветке origin/develop.

Каждый разработчик работает в своей ветке типа origin/feature/portfolio-"имяучастника"
